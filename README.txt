
# App WhatsApp Streamlit

Esta aplicación simple permite que cualquier usuario ingrese su nombre y un mensaje,
y al hacer clic en "Enviar a WhatsApp", se redirige automáticamente al número +56967010107.

## Instrucciones para ejecutar

1. Instala Streamlit si no lo tienes:
   pip install streamlit

2. Ejecuta la aplicación:
   streamlit run app.py
